- **JWT** - Authentication tokens
- **OAuth 2.0** - Third-party login

- **React 18** - UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool
- **TailwindCSS** - Utility-first styling
- **LiveKit React Components** - Video UI
